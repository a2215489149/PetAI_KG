
這是一個基於增強 LightRAG/GraphRAG（圖譜增強檢索） 與多模態動態學習 的「自我演化寵物照護 AI」專案。

🐾 專案描述：PetAI 自我演化寵物 RAG 系統
這套系統不僅是一台會說話的機器人，而是一個具備「長期記憶」與「專家知識」的寵物健康顧問。它能透過 LINE 介面，在群組或私訊中精確識別寵物的行為、外觀，並不斷透過對話學習每一隻寵物的獨特個性與病史。

🌟 核心技術亮點
多模態動態感知：蒸餾 Gemini 2.5 Flash至本地端VLM，具備實時圖像與影音分析能力，能解讀寵物的姿態、動作與環境細節，並將觀察結果靜默注入對話背景。
自我演化知識圖譜 (GraphRAG)：
動態錨定 (Dynamic Anchoring)：AI 會從對話中自動提取「實體-關係-摘要」三元組，病史、喜好、年齡等資訊會即時存入 Neo4j。
權威知識匯入 (LightRAG)：結合專業寵物醫療文獻，建構超過 50 個「超節點 (SuperNodes)」的權威分類體系。
群組 Session 同步技術：創新實作了 Session ID 邏輯，在 LINE 群組中實現跨成員的「上下文同步」與「媒體觀察共享」，讓群組內所有成員都能與同一個 AI 大腦進行連貫對話。
雙重檢索機制：
Qdrant 向量檢索：利用語意相似度撈取相關的醫學事實與過往代對話碎。
Neo4j 結構化檢索：精確過濾個人專屬的寵物檔案數據。
高性能架構：採用 FastAPI 異步框架，搭配 Redis 處理高併發的消息鎖定（Locking）與 20 則訊息的滾動短期記憶裁切。
🎯 應用場景
多貓/多狗家庭：自動建立多隻寵物的獨立檔案。
醫療追蹤：記錄寵物什麼時候打了疫苗、最近的排泄狀況，並在需要時提供專業的醫療指導。
行為觀察：透過影片分析寵物的步態或情緒，提供照護建議。
這是一個將 LLM 應用、向量數據庫、圖論數據庫 與 行動通訊介面 完美整合的現代 AIGC 落地工程實驗。
